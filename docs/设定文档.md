# 设定文档

## ID分配准则

- 1000 - 1999：基础设定层

  - 1000-1099: 种族 (RaceData)

  - 1100-1199: 职业 (JobData)

- 2000 - 2999：动作原子层 (EffectData)

  - 2000-2199: 属性改变类（加血、扣蓝等）

  - 2200-2399: 状态增减类（施加、驱散 Buff）

- 3000 - 4999：持续状态层 (BuffData)

  - 3000-3999: 增益 Buff (Buffs)

  - 4000-4999: 减益 Buff (Debuffs)

- 5000+：主动技能层 (SkillData)
